# Financial Control Automation

Python-based automation project designed to manage financial obligations and reminders through WhatsApp notifications and Google Sheets integration.

## Features

- WhatsApp notifications
- Automatic reminders
- Weekly summaries
- Google Sheets integration
- Webhooks
- Scheduled tasks
- Environment variables
- Logging system

## Tech Stack

- Python
- Google Sheets API
- Webhooks
- WhatsApp API
- Cron / Scheduler

## Architecture

```
WhatsApp
    ↓
Python
    ↓
Processing
    ↓
Google Sheets
    ↓
Notifications
```

## Project Structure

```
main.py
parser.py
webhook.py
whatsapp.py
folhas.py
requirements.txt
```

## Goals

- Reduce manual work
- Centralize financial information
- Generate reminders automatically
- Improve reliability and organization
